The passed value is compared to the current string, and if they are equal, the other value that was passed in is returned. If they are already different, the first value passed in is returned. 注意：这个方法返回新值，但并不改变现有字符串。Note that this method returns the new value but does not change the current string. 

// 可供选择的排序方向。alternate sort directions

// 等价判断语句：instead of conditional logic:
sort = (sort == 'ASC' ? 'DESC' : 'ASC');
