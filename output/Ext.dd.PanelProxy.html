// Internal developer documentation -- will not show up in API docs 为Ext.Panel对象定制的拖拽代理的实现。该主要被Panel对象在内部调用以实现拖拽，因此不需要直接创建。A custom drag proxy implementation specific to Ext.Panels. This class is primarily used internally for the Panel's drag drop implementation, and should never need to be created directly.
