类Ext.data.ArrayReader

包： Extdata
定义的文件： ArrayReader.js
类全称： Ext.data.ArrayReader
继承自于： Ext.data.JsonReader

Data reader透过一个数组而转化成为Ext.data.Record对象所组成的数组，数组内的每个元素就代表一行数据。字段定义中，通过使用下标来把字段抽取到Record对象，属性mapping用于指定下标，如果不指定就按照定义的先后顺序。
Data reader class to create an Array of Ext.data.Record objects from an Array. Each element of that Array represents a row of data fields. The fields are pulled into a Record object using as a subscript, the mapping property of the field definition if it exists, or the field's ordinal position in the definition.

示例代码： Example code:. 

var Employee = Ext.data.Record.create([
    {name: 'name', mapping: 1},        // 属性mapping用于指定下标 // "mapping" only needed if an "id" field is present which
    {name: 'occupation', mapping: 2}   // 如果不指定就按照定义的先后顺序 // precludes using the ordinal position as the index.
]);
var myReader = new Ext.data.ArrayReader({
    id: 0                 // 提供数组的下标位置存放记录的ID（可选的）    // The subscript within row Array that provides an ID for the Record (optional)
}, Employee);

形成这种形式的数组：This would consume an Array like this: 


[ [1, 'Bill', 'Gardener'], [2, 'Ben', 'Horticulturalist'] ]
