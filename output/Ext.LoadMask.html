类Ext.LoadMask

包： Ext
定义的文件： LoadMask.js
类全称： Ext.LoadMask
继承自于： Object
一个简单的工具类，用于在加载数据时为元素做出类似于遮罩的效果。对于有可用的Ext.data.Store，可将效果与Store的加载达到同步，而mask本身会被缓存以备复用。而对于其他元素，这个遮照类会替换元素本身的UpdateManager加载指示器，并在初始化完毕后销毁。A simple utility class for generically masking elements while loading data. If the store config option is specified, the masking will be automatically synchronized with the store's loading process and the mask element will be cached for reuse. For all other elements, this mask will replace the element's Updater load indicator and will be destroyed after the initial load.

用法举例：Example usage:


// Basic mask:
var myMask = new Ext.LoadMask(Ext.getBody(), {msg:"Please wait..."});
myMask.show();
