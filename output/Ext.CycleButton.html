类Ext.CycleButton

包： Ext
定义的文件： CycleButton.js
类全称： Ext.CycleButton
继承自于： Ext.SplitButton
一个包含Ext.menu.CheckItem元素的特殊分割按钮。按钮会在点击时自动循环选中每个菜单项，并以该项为活动项触发按钮的change事件（或者调用按钮的changeHandler函数，如果设置过的话）。通过点击箭头区域即可像普通分割按钮那样显示下拉列表。使用示例: A specialized SplitButton that contains a menu of Ext.menu.CheckItem elements. The button automatically cycles through each menu item on click, raising the button's change event (or calling the button's changeHandler function, if supplied) for the active menu item. Clicking on the arrow section of the button displays the dropdown menu just like a normal SplitButton. Example usage: 

var btn = new Ext.CycleButton({
    showText: true,
    prependText: 'View as ',
    items: [{
        text:'text only',
        iconCls:'view-text',
        checked:true
    },{
        text:'HTML',
        iconCls:'view-html'
    }],
    changeHandler:function(btn, item){
        Ext.Msg.alert('Change View', item.text);
    }
});
