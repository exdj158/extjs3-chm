类Ext.layout.AbsoluteLayout

包： Extlayout
定义的文件： AbsoluteLayout.js
类全称： Ext.layout.AbsoluteLayout
继承自于： Ext.layout.AnchorLayout

此类是Ext.layout.AnchorLayout的子类，它增加了通过标准的x、y的配置项来定位的能力。
Inherits the anchoring of Ext.layout.AnchorLayout and adds the ability for x/y positioning using the standard x and y component config options.

应通过继承此类或设置Ext.Containerlayout的配置layout:'absolute'的方式创建该类，一般很少通过关键字new直接使用该类。
This class is intended to be extended or created via the layout configuration property. See Ext.Containerlayout for additional details.

用法举例：Example usage:


var form = new Ext.form.FormPanel({
    title: 'Absolute Layout',
    layout:'absolute',
    layoutConfig: {
        // 布局的配置项就在这里配置 layout-specific configs go here
        extraCls: 'x-abs-layout-item'
    },
    baseCls: 'x-plain',
    url:'save-form.php',
    defaultType: 'textfield',
    items: [{
        x: 0,
        y: 5,
        xtype:'label',
        text: 'Send To:'
    },{
        x: 60,
        y: 0,
        name: 'to',
        anchor:'100%'  // 由百分比来定位宽度 anchor width by percentage
    },{
        x: 0,
        y: 35,
        xtype:'label',
        text: 'Subject:'
    },{
        x: 60,
        y: 30,
        name: 'subject',
        anchor: '100%'  // 由百分比来定位宽度 anchor width by percentage
    },{
        x:0,
        y: 60,
        xtype: 'textarea',
        name: 'msg',
        anchor: '100% 100%'  // 由百分比来定位高、宽 anchor width and height
    }]
});

  • Box Layout : 'x-box-item'
