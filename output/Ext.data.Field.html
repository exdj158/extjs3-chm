类Ext.data.Field

包： Extdata
定义的文件： DataField.js
类全称： Ext.data.Field
继承自于： Object

该类封装了Ext.data.Recordcreate所传入字段定义对象的信息。
This class encpasulates the field definition information specified in the field definition objects passed to Ext.data.Recordcreate.

开发者一般不需要实例化该类。方法执行过程中创建该实例，并且在Record构造器的原型prototype中的fields属性就是该实例。
Developers do not need to instantiate this class. Instances are created by Ext.data.Record.create and cached in the fields property of the created Record constructor's prototype.
