  defaultMargins : Object
设置外边距（margins）。如果不制定每个子项的margins属性，...

设置外边距（margins）。如果不制定每个子项的margins属性，那么就会使用默认的margins。If the individual contained items do not have a margins property specified, the default margins from this property will be applied to each item.


This property may be specified as an object containing margins to apply in the format:指定margin的格式如下：


{
    top: (top margin),
    right: (right margin),
    bottom: (bottom margin),
    left: (left margin)
}

This property may also be specified as a string containing space-separated, numeric margin values. The order of the sides associated with each value matches the way CSS processes margin values:该属性接纳的格式也可以是字符串的，其格式规定为空格隔开，数字类型的margin值。各个方向的排列顺序为CSS的顺序：

  • 如果只是一个值，就是各个方向都是这个值。If there is only one value, it applies to all sides.
  • 如果有两个值，那么第一个值代表是上下方向的值，第二值则是左右方向的值。If there are two values, the top and bottom borders are set to the first value and the right and left are set to the second.
  • 如果有三个值，第一个值是顶部的值，第二个值是左右两边的值，第三个值是底部的值。If there are three values, the top is set to the first value, the left and right are set to the second, and the bottom is set to the third.
  • 如果有四个值，分别代表就是顶部、右边、底部和左边的值（顺时针方向）。If there are four values, they apply to the top, right, bottom, and left, respectively.

默认为：Defaults to:

 {top:0, right:0, bottom:0, left:0}
